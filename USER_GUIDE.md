# User Guide

## Open a course

1. Launch SYNAPSE.
2. Click a base card on the homepage.
3. Explore the knowledge graph and double-click a node to open its topic center.

## Work inside a topic

- Edit notes in the `Notes Editor`
- Track exercises in `Practice Tracker`
- Keep formulas in `Formula Vault`
- Organize tasks in `Kanban Board`
- Review images, resources, PDFs, and timeline entries
- Drag blocks by the top handle and resize them from the lower-right corner

## Quick capture

- Use `Ctrl+Shift+Space` to open Quick Capture
- Capture files, screenshots, notes, or links into a node
- Hot-drop files into the configured hot-drop folder to route them into the active node

## Graph basics

- Click a node to select it
- Double-click an unlocked node to open it
- Use arrow keys to move between connected nodes
- Press `0` to zoom to fit
- Toggle filters from the left drawer

## Exam prep

- Enable exam prep mode from the command palette
- The sidebar prioritizes high-weight, low-mastery topics
- Countdown and study-plan suggestions appear when an exam date exists in `syllabus.json`

## Analytics

- Open Settings
- Launch the analytics dashboard
- Review readiness, mastery distribution, study activity, and top-priority topics
- Export a CSV snapshot for external review

## Git workflow

- Use the footer sync button to pull and push the current course repo
- Use manual commit when you want a named checkpoint
- Enable auto-commit in settings to save routine changes automatically
