# Privacy Policy

SYNAPSE is designed as a local-first desktop application.

## What SYNAPSE stores

- Local configuration files in the operating system application-data directory
- Course content that you place inside your own learning repository
- Optional Git history in repositories you choose to initialize or sync

## What SYNAPSE does not do by default

- No cloud sync service
- No telemetry
- No external analytics
- No scraping
- No AI requests

## Update checks

If update checking is configured by the app distributor, SYNAPSE may contact the configured update endpoint to check for new releases. No course content is uploaded as part of a normal update check.

## User control

- You control the location of your learning repository
- You can disable Git-related behavior in settings
- You can inspect all persisted course data because it is file-based
