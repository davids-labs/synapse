# Shortcuts

- `Ctrl+Shift+Space`: open Quick Capture
- `Ctrl+K`: open Command Palette
- `Ctrl+H`: return home
- `Ctrl+S`: sync current course
- `Ctrl+B`: toggle filter drawer
- `0`: zoom graph to fit
- `F`: toggle focus mode
- `Enter`: open selected topic
- `Escape`: close topic center or overlays
- `Arrow Keys`: move between connected nodes on the graph
